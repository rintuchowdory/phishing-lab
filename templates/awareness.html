⚠️ Security Awareness Training

This was a phishing simulation. You entered credentials into a fake login page.

What went wrong?

  • You trusted the page without verifying the URL
  • No MFA was used
  • The page looked legitimate

How to protect yourself

  • Always check the website address
  • Use a password manager
  • Enable Multi-Factor Authentication (MFA)
  • Report suspicious emails
